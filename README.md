## Commands to execute/test/etc
After adding gems
```bash
bundle install
```
Updating gems
```bash
bundle update
```
Running the blog local
```bash
bundle exec jekyll serve
```

Previewing an open PR in Heroku before merging
```bash
heroku ps:scale web=1 --app <name of the preview app>
```
